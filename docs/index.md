## 产品概述

托福备考助手是一款面向托福考生的智能备考平台，覆盖阅读、听力、口语、写作四大科目，集成 AI 题目解析、智能陪练、自适应学习规划与后台数据分析。

**核心目标**：帮助考生高效备考，通过智能练习和个性化学习计划提升托福成绩。

**适用人群**：
- 准备参加托福考试的考生
- 需要系统化英语学习的学生
- 希望提升英语能力的自学者

## 功能特性

### 📚 四科练习
- **阅读**：文章阅读 + 题目练习 + 计时 + 结果分析
- **听力**：音频播放 + 题目练习 + 倍速播放
- **口语**：录音答题 + AI 评分（预留）
- **写作**：在线编辑器 + 计时 + AI 批改（预留）

### 🧠 智能学习
- **AI 陪练**：口语对话 / 听力沉浸模式，实时评分反馈
- **自适应学习计划**：基于目标分数和考试日期，自动生成分阶段学习计划
- **遗忘曲线复习**：改进版 SM-2 算法，智能推送错题复习
- **全真模拟**：四科连考模式，还原真实考试流程

### 💼 商业化（预留）
- **会员体系**：免费用户 / 付费会员，功能分级
- **支付系统**：微信支付接入（预留），支持月卡/季卡/年卡
- **广告系统**：首页 Banner、练习完成页、错题本广告位
- **数据看板**：管理后台用户增长、活跃度、付费转化统计

## 技术架构

### 技术栈概览

| 层次 | 技术 | 说明 |
|------|------|------|
| 小程序前端 | 微信小程序原生 (WXML/WXSS/JS) | 原生性能，直接调用微信 API |
| Web 前端 | Vue 3 + Vite + Element Plus | 响应式设计，适配移动端 |
| 管理后台 | Vue 3 + ECharts + Element Plus | 数据可视化 |
| 后端 | Node.js + Express | RESTful API，JWT 鉴权 |
| 数据库 | PostgreSQL 16 | 关系型 + JSONB，适合题库存储 |
| 缓存 | Redis | 会话管理 + 限流 |
| AI 服务 | OpenAI API（预留） | AI 评分 + 陪练对话 |
| 部署 | Render.com + GitHub Pages | 后端免费托管 + 前端静态部署 |

## 快速开始

### 前置条件
- Node.js >= 18.0.0
- PostgreSQL >= 16.0
- npm >= 9.0

### 1. 克隆项目
```bash
git clone https://github.com/huieric/toefl-miniapp.git
cd toefl-miniapp
```

### 2. 启动后端
详细步骤请参考 [部署文档](./deployment.md#后端部署)

### 3. 启动 Web 前端
```bash
cd web
npm install
npm run dev
```

### 4. 小程序预览
1. 打开 **微信开发者工具**
2. 导入项目，选择 `miniprogram/` 目录
3. 填入自己的 `AppID`（测试可使用测试号）
4. 修改 `miniprogram/utils/config.js` 中的 API 地址

## 常见问题

### Q1: 如何体验小程序？
A: 目前小程序需要微信审核发布。您可以通过以下方式体验：
1. **Web 端**：访问 [https://huieric.github.io/toefl-miniapp/web/](https://huieric.github.io/toefl-miniapp/web/)
2. **本地开发**：使用微信开发者工具导入 `miniprogram/` 目录
3. **测试号**：申请微信小程序测试号进行体验

### Q2: 需要付费吗？
A: 基础功能免费，AI 陪练和高级功能需要会员订阅。

### Q3: 支持哪些设备？
A: 支持微信小程序（手机端）、Web 浏览器（PC/手机）、管理后台（PC）。

### Q4: 数据安全吗？
A: 所有用户数据加密存储，遵循 GDPR 和隐私保护规范。

### Q5: 如何反馈问题？
A: 通过 [GitHub Issues](https://github.com/huieric/toefl-miniapp/issues) 提交问题。
